# QR Code Generator

A web application that allows users to generate QR codes from text, URLs, or messages.  
The system also includes user authentication and an admin panel to manage QR code generation history.

---

## Features

- User Registration and Login
- Generate QR Codes from Text or URL
- Customize QR Code Size and Color
- Download Generated QR Code
- Admin Dashboard to View QR Code History
- Data stored in MySQL database

## How to Run the Project

1. Install XAMPP on your system.
2. Copy the project folder into:

C:\xampp\htdocs

3. Start Apache and MySQL in XAMPP.
4. Open phpMyAdmin and import the database file.
5. Open your browser and run:

http://localhost/qr_code_generator
