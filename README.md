QR Code Generator
A web application that allows users to generate QR codes from text, URLs, or messages. The system also provides user authentication and an admin dashboard to view QR code generation history.
Features
User Registration and Login
Generate QR Codes from Text or URL
Customize QR Code Size and Color
Download Generated QR Code
Admin Dashboard to View History
Secure Database Storage
Technologies Used
HTML
CSS
JavaScript

MySQL

How to Run the Project
Install XAMPP
Move the project folder to:
C:\xampp\htdocs
Start Apache and MySQL
Import the database file in phpMyAdmin
Open the browser and run:
http://localhost/qr_code_generator
